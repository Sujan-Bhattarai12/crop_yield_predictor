#  Crop Forecasting App

This application provides predictive insights into crop yields across 10 different countries using historical datasets and machine learning techniques. With a dataset of over **10,000 observations**, the app helps stakeholders — including farmers, policymakers, and researchers — make informed agricultural decisions.

##  Features

- Forecasts crop production using **LightGBM**, a high-performance gradient boosting model.
- Implements **ARIMA** time series modeling for robust temporal forecasting.
- Supports multi-country and multi-crop datasets for scalable analysis.
- Visualizations and tabular outputs to compare actual vs. predicted values.

##  Dataset

- 10,000+ observations  
- Covers approximately 10 countries  
- Includes variables like crop type, climate metrics, historical yields, and time series indices

##  Models Used

- **LightGBM** for feature-rich tabular data modeling  
- **ARIMA** for univariate time series forecasting
